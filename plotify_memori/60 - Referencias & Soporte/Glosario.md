## Termino de negocio

| Termino | Definicion |
|---------|-----------|
| **Lote** | Parcela individual dentro de un proyecto de loteo |
| **Loteo** | Subdivision de un terreno en lotes |
| **Reserva** | Accion de apartar un lote para un comprador (paso previo a la venta) |
| **Escritura** | Documento legal de transferencia de propiedad (escritura publica) |
| **Deslinde** | Descripcion legal de los limites de un lote con sus vecinos |
| **Servidumbre** | Derecho de paso por un lote para acceder a otros (servidumbre de transito) |
| **SAG** | Servicio Agricola y Ganadero (organismo regulatorio en Chile) |
| **CBR** | Conservador de Bienes Raices (registro de propiedades) |
| **Comparecientes** | Personas que firman una escritura publica |
| **Matriz** | Propiedad original antes de la subdivision |
| **Vendedor/Vendor** | Agente de venta asignado a un proyecto |

## Termino tecnico

| Termino | Definicion |
|---------|-----------|
| **KMZ** | Formato de Google Earth (ZIP que contiene KML) |
| **KML** | Keyhole Markup Language (formato XML para datos geoespaciales) |
| **GeoJSON** | Formato JSON para datos geograficos |
| **UTM** | Universal Transverse Mercator (sistema de coordenadas en metros) |
| **WGS84** | World Geodetic System 1984 (lat/lng para mapas web) |
| **RLS** | Row Level Security (seguridad a nivel de fila en PostgreSQL) |
| **RPC** | Remote Procedure Call (funciones almacenadas en la DB) |
| **MCP** | Model Context Protocol (conexiones a servicios externos para el agente IA) |
| **RSC** | React Server Components |
| **SSR** | Server-Side Rendering |
| **LangGraph** | Framework de grafos de estado para agentes IA |
| **arq** | Cola de tareas async con Redis para Python |
| **Jinja2** | Motor de templates de Python (usado para documentos) |
| **shadcn/ui** | Libreria de componentes UI accesibles |
| **Turf.js** | Libreria de analisis geoespacial para JavaScript |
| **MapLibre GL** | Libreria de mapas interactivos (fork open-source de Mapbox GL) |
| **proj4** | Libreria de proyecciones cartograficas |
